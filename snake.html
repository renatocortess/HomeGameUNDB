Instruções do Jogo

Coloque a tela como melhor ficar para você, clique no quadrado preto da Snake, depois movimente a Snake com os comandos:

  • A -> ESQUERDA
  • W -> CIMA
  • S -> PARA BAIXO
  • D -> DIREITA

PARA RESPONDER AS PERGUNTAS, DIGITE v para verdadeiro e f para falso.

Bom jogo!
